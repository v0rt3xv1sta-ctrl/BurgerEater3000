# Burger Eater 3000

A simple burger clicker game made in Python with Tkinter.  
Click "Eat Burger" to earn burgers. Spend 20 burgers to unlock the automatic bot that earns burgers for you!

## How to play
1. Run `python3 BurgerEater3000.py`
2. Click "Eat Burger" to earn burgers
3. When you have 20 burgers, click "Unlock Bot" to start automatic burger collection
